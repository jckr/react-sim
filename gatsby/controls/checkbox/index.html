Checkbox

A Checkbox is a simple control that will cause a param to toggle from true to false. It doesn't need anything else besides the common props.

<Model
  showTimer={false}
  initialParams={{ checked: false }}
  controls={{
    type: 'checkbox',
    label: 'Check me',
    param: 'checked',
  }}
/>
