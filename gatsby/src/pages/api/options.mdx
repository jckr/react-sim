| Name                                 | Description                                                                                                                                                                                                                                                                                                                                     |
| ------------------------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **options**<br/>array option objects | options is an array of option objects, which have _label_ and _value_ properties. _label_ is a string, _value_ can be a string, a number or a boolean. _label_ is what is displayed, _value_ is the value that will be passed when the control changes. If _label_ and _value_ are identical, instead of an object a simple string can be used. |
