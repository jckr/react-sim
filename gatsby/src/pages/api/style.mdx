### Style

`Model` uses the [rebass](https://rebassjs.org/) component system.
Rebass is part of a larger ecosystem that includes [theme-ui](https://theme-ui.com/) and the [styled system](https://styled-system.com/).

As such, `Model` can take a theme-ui `theme` and styled system `sx` props.

| Name                           | Description                                               |
| ------------------------------ | --------------------------------------------------------- |
| **theme**<br/>object, optional | A [theme-ui](https://theme-ui.com/) theme.                |
| **sx**<br/>object, optional    | A [styled system](https://styled-system.com/) style prop. |
