All the [connected Frame](/api/#withframe) context props, plus:

| Name                            | Description                                                                                   |
| ------------------------------- | --------------------------------------------------------------------------------------------- |
| **draw**<br/>function, optional | A function which will be called each time CanvasFrame is updated (ie at each animation cycle) |
| **height**<br/>number, optional | The height of the canvas. If ignored, CanvasFrame will used `params.height`.                  |
| **width**<br/>number, optional  | The width of the canvas. If ignored, CanvasFrame will used `params.width`.                    |
