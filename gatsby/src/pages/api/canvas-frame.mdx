All the [connected Frame](/api/#withframe) context props, plus:

|Name|Description|
|----|-----------|
|__draw__<br/>function, optional|a function which will be called each time CanvasFrame is updated (ie at each animation cycle)|
|__height__<br/>number, optional|the height of the canvas| If ignored, CanvasFrame will used `params.height`.
|__width__<br/>number, optional|the width of the canvas| If ignored, CanvasFrame will used `params.width`.
