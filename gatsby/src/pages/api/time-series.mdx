| Name                                    | Description                                                                                                                                                                    |
| --------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **series**<br/>series object or array   | Either a single [series](/api/#series) object or an array of series. the accessor function is going to be called on every item in `cachedData` and the result will be plotted. |
| **stacked**<br/>boolean, default: false | If true, represents the time series as full-height, stacked area charts. If false, the series are line charts.                                                                 |
| **height**<br/>number, optional         | Height of the chart.                                                                                                                                                           |
| **padding**<br/>number, optional        | Padding of the chart.                                                                                                                                                          |
| **width**<br/>number, optional          | Width of the chart.                                                                                                                                                            |
