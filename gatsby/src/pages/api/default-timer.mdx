| Name                       | Description                                                                                                       |
| -------------------------- | ----------------------------------------------------------------------------------------------------------------- |
|__minTime__<br/>integer, 0 by default|at which position the timer starts. Can be negative.|
|__maxTime__<br/>integer, 100 by default|at which position the timer ends.|
|__showTime__<br/>boolean, true by default|Whether the time block (i.e. slider and time value) is going to be shown.|
|__showTimeSlider__<br/>boolean, true by default|Whether the time slider is going to be shown.|
|__showTimer__<br/>boolean, true by default|Whether the Timer is shown at all. If the Timer is not shown, you will have to provide another way to start the simulation (or effectively force the user to stay on the initial state).|
