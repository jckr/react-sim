`series` is an object with three properties:

| Name                      | Description                                                             |
| ------------------------- | ----------------------------------------------------------------------- |
| **label**<br/>string      | Describes the series,                                                   |
| **color**<br/>string      | Color of the series,                                                    |
| **accessor**<br/>function | Function that turns a data object into the value that you want to plot. |
