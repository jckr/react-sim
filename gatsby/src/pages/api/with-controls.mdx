| Name                       | Description                                                                                                       |
| -------------------------- | ----------------------------------------------------------------------------------------------------------------- |
| **isPlaying**<br/>boolean  | tells your custom control if the simulation is playing.                                                           |
| **params**<br/>object      | any and all of the simulation parameters.                                                                         |
| **pause**<br/>function     | call this function to _pause_ the simulation (it stops playing, but is not reset.)                                |
| **play**<br/>function      | call this function to start or resume playing the simulation.                                                     |
| **stop**<br/>function      | call this function to stop the simulation (stops playing, time is reset to default, data of simulation is reset.) |
| **updateTime**<br/>fuction | call with a number to update the `tick` value in the simulation.                                                  |
