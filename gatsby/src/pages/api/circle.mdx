`circle` takes an object as argument with the following properties:

All the [CanvasFrame](/api/#canvasframe) properties, plus:

| Name             | Description                   |
| ---------------- | ----------------------------- |
| **x**<br/>number | x coordinate of circle center |
| **y**<br/>number | y coordinate of circle center |
| **r**<br/>number | Radius of circle              |
