Its arguments are of the form:
`({cachedData, complete, data, params, pause, stop})`. Its output will replace `data` in the state.

| Name                      | Description                                                                                                                          |
| ------------------------- | ------------------------------------------------------------------------------------------------------------------------------------ |
| **data**<br/>any          | Represents the current value of data.                                                                                                |
| **tick**<br/>integer      | Represents the new tick.                                                                                                             |
| **params**<br/>object     | Represents the current parameters.                                                                                                   |
| **cachedData**<br/>object | An object so that for each past value of `tick`, `cachedData[tick]` represents the value of `data` at that moment of the simulation. |
| **complete**<br/>function | A function that signals the simulation is complete (it will also stop).                                                              |
| **pause**<br/>function    | A function that will stop the simulation from automatically updating and animating.                                                  |
| **stop**<br/>function     | A function that will stop the simulation from automatically updating and animating, and will reset it.                               |
