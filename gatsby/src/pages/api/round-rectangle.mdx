`roundRectangle` takes an object as argument with the following properties:

All the [CanvasFrame](/api/#canvasframe) properties, plus:

| Name                        | Description                                  |
| --------------------------- | -------------------------------------------- |
| **x**<br/>number            | x coordinate of top-left corner of rectangle |
| **y**<br/>number            | y coordinate of top-left corner of rectangle |
| **width**<br/>number        | Width of rectangle                           |
| **height**<br/>number       | Height of rectangle                          |
| **tl**<br/>number, optional | Radius of top-left corner                    |
| **tr**<br/>number, optional | Radius of top-right corner                   |
| **bl**<br/>number, optional | Radius of bottom-left corner                 |
| **br**<br/>number, optional | Radius of botrom-right corner                |
| **r**<br/>number, optional  | Radius of all unspecified corners            |
