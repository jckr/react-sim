# Timer

You can also explicitly specify your timer, so it can be anywhere you want in the controls, not just at the very end (default).

<TimerExample />

The Timer takes the following props:

- label (string, 'Time' by default) - The label that will go above the slider. Note that the [default timer](/controls/default-timer) doesn't let you edit the label.
- name (string, 'Time' by default) - the name of the control.
- minTime (integer, 0 by default) - minimum time.
- maxTime (integer, 100 by default) - maximum time.
- showTime (boolean, true by default) - whether we are going to show the slider part of the timer (along with label, min and max values)
- showTimeSlider (boolean, true by default) - whether we are going to show the slider, min and max time values.
- all the range props.

```jsx
<Model
  showTimer={false}
  controls={{
    type: 'timer',
    label: 'My explicit timer',
  }}
/>
```

If you explictly include a `timer` through the `controls` prop of `Model`, then the [default timer](/controls/default-timer) won't be added at the end.
