# Select

Select is a drop-down menu with a limited number of options.

<SelectExample />

Select takes the following props:

<SelectApi />

```jsx
<Model
  showTimer={false}
  initialParams={{ options: 'jet' }}
  controls={[
    {
      type: 'select',
      label: 'Choose one',
      param: 'options',
      options: ['jet', 'set', 'radio'],
    },
  ]}
/>
```
