# Radio

A radio control will create several radio buttons, one of which is selected.

<RadioExample />

In addition to the [Common Control Props.](/api/#common-controls-properties), `radio` takes:

- options (array of strings): List of the values that can be chosen.
- vertical (boolean, false by default): whether the options are displayed down (true) or across (false, default).

```jsx
<Model
  showTimer={false}
  initialParams={{ options: 'jet' }}
  controls={[
    {
      type: 'radio',
      label: 'Choose one',
      param: 'options',
      options: ['jet', 'set', 'radio'],
    },
  ]}
/>
```
